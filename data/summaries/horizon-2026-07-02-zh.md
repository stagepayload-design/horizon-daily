## [首个从头构建的合成细胞实现生长与分裂](https://www.quantamagazine.org/for-the-first-time-a-cell-built-from-scratch-grows-and-divides-20260701/) ⭐️ 9.0/10

由 Kate Adamala 领导的 Biotic 团队创造了 SpudCell，这是首个完全从头构建、无需细胞骨架即可生长和分裂的合成细胞。该细胞利用脂质合成驱动的膜管化和断裂机制进行分裂。 这一突破证明，最小的合成细胞可以实现完整的生命周期，使合成生物学更接近创造真正的人工生命。它可能通过设计具有定制功能的细胞，在医学、材料和环境应用领域引发生物技术革命。 SpudCell 的基因组仅包含 541 个基因，远少于典型细菌的 4000 多个基因。该细胞依赖化学定义的培养基，不使用细胞骨架；分裂通过脂质合成驱动的膜管化和断裂过程实现。

hackernews · defrost · Jul 1, 14:20 · [社区讨论](https://news.ycombinator.com/item?id=48747304)

**背景**: 合成生物学旨在从非生命组分构建人工细胞，以理解生命的最低要求。以往的合成细胞能够生长和复制 DNA，但无法分裂，因为分裂通常需要复杂的细胞骨架。SpudCell 通过使用更简单的基于脂质的机制绕过了这一障碍。

Cloudflare 宣布推出 Monetization Gateway，该平台利用 x402 协议，通过稳定币支付对任何资源（API、数据集、网页或 AI 工具）进行收费，为 AI 代理和用户实现即时微交易。 这解决了网络上长期存在的无缝微交易需求，使 AI 代理无需手动管理 API 密钥即可付费访问，并可能为内容创作者和 API 提供商解锁新的收入模式。 Monetization Gateway 利用 Cloudflare 遍布 330 多个城市的全球网络，在靠近买家的位置执行 x402 握手，从而降低延迟。它使用稳定币进行结算，绕过了传统的支付通道。
