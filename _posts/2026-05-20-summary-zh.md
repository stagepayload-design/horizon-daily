## [Forge 将 8B 模型准确率从 53% 提升至 99%](https://github.com/antoinezambelli/forge) ⭐️ 8.0/10

Forge 是一个面向自托管 LLM 的开源可靠性层，通过重试提示和错误恢复等护栏机制，将 8B 模型在多步骤智能体任务上的准确率从 53% 提升至 99.3%，达到前沿 API 水平。 这大幅缩小了本地模型与昂贵云端 API 之间的可靠性差距，使得在消费级硬件上运行经济实惠、始终在线的智能体系统成为可能。 Forge 的护栏堆栈包含五个可独立开关的层，其中重试提示和错误恢复带来的提升最大；搭载 Forge 的 8B 模型（99.3%）优于未加护栏的 Claude Sonnet（87.2%）。

hackernews · zambelli · May 19, 12:23 · [社区讨论](https://news.ycombinator.com/item?id=48192383)

**背景**: 多步骤智能体任务要求 LLM 顺序调用工具，每一步的错误会累积。即使单步准确率达 90%，5 步后成功率也仅约 59%。Forge 通过添加重试逻辑、错误分类（如 ToolResolutionError）和 VRAM 感知的上下文管理来解决这一问题。
